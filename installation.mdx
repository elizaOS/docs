## One-Command Install

Get elizaOS running with a single command:

```bash
bun i -g @elizaos/cli && elizaos create my-agent
```

<Note>
This installs the CLI globally and creates your first agent project in one step.
**No additional setup required** - the CLI handles dependencies automatically.
</Note>

## Prerequisites

<AccordionGroup>
  <Accordion title="Node.js & Bun" icon="cube" defaultOpen>
    elizaOS requires **Node.js 23.3+** and **Bun** for optimal performance.
    
    **Quick Install (Mac/Linux):**
    ```bash
    # Install Node.js via nvm
    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
    source ~/.zshrc
    nvm install 23.3 && nvm use 23.3
    
    # Install Bun
    curl -fsSL https://bun.sh/install | bash
    source ~/.zshrc
    ```
    
    **Windows Users:**
    - Install [WSL2](https://learn.microsoft.com/en-us/windows/wsl/install) (recommended)
    - Or use [Git Bash](https://git-scm.com/downloads/win) with native Windows
    
    **Verify Installation:**
    ```bash
    node --version  # Should show v23.3.0+
    bun --version   # Should show version number
    ```
  </Accordion>

## Verify Installation

After installation, verify everything works:

```bash
# Check CLI installation
elizaos --version

# Create a test project
elizaos create test-agent

# Start the agent
cd test-agent && elizaos start
```

<Check>
**Success!** If you see the elizaOS interface at `http://localhost:3000`, you're ready to build agents.
</Check>
